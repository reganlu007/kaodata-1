### Notes

* Steam 版 SAN1, SAN2 用的是 PC9801 的版本
* SAN1 DOS 版(Eng.) 與 Steam/PC98 版，除了顏色的差別外(4色 vs 8色), 顏良頭盔也是一個差別
* 上面兩個版本在大眾臉武將處的空白也有差異
* SAN2 DOS 版 與 Steam/PC98 版，人物幾乎重繪
